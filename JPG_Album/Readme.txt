JPGAlbumCreator - для создания фотоальбома
в формате  *.jpga  из  отдельных  jpeg-снимков.

JPGAlbumRider  - для просмотра  фотоальбома
в формате  *.jpga. При запуске данный формат 
регистрируется  в реестре Windows. В  других
программах   просмотра   изображений  виден
только первый снимок.
